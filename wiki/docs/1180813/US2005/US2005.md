# User Story #2005=======================================# 1. Requisitos**US2005**Esta User Story consiste em importar o catálogo de produtos através de um ficheiro CSV, enquanto Gestor de Produção.# 2. AnálisePara esta funcionalidade será necessária uma autenticação prévia como Gestor de Produção (GP), já que este utilizador deve ter acesso ao catálogo de produtos, para, neste caso, o alterar.# 3. Design## 3.1. Realização da Funcionalidade1. O Gestor de Produção inicia a importação do catálogo de produtos.2. O sistema solicita o nome do ficheiro com o catálogo.3. O Gestor de Produção introduz o nome do ficheiro solicitado.4. O sistema mostra o nome introduzido e solicita confirmação.5. O Gestor de Produção confirma o nome introduzido.6. O sistema importa o catálogo e comunica o sucesso da operação.![SD_US2005](SD_US2005.png)## 3.2. Diagrama de Classes![CD_US2005](CD_US2005.png)## 3.3.
